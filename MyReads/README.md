# MyReads Project

## Introduction

This is a bookshelf React app that allows the selection and categorisation of books that are already read, are currently being read, or wanting to be read.

Search allows for books to be found that can be added to the bookshelf.

## Instructions

* Install all project dependencies with `npm install`.
* Start the server with `npm start`.

The backend API uses a fixed set of cached search results and is limited to a particular set of search terms. The book search options available with the API are as follows:
