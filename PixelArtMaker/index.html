Lab: Pixel Art Maker

This 'Pixel Art Maker' lets you draw on a canvas size that you define.

Select the size of the canvas grid that you want to draw on by choosing the height and width values (Note: Canvas max size is 50x50) and then clicking submit. This will create the canvas under the 'Design Canvas' section. Select the colour of choice from the 'Pick a Colour' section and draw by clicking on the canvas grid (either individual cell or multiple at a time, by holding the mouse button).

If you accidently colour a cell with the wrong colour, double click that cell to erase it.

There is an options to add/remove a height/width to the canvas, after the initial canvas is created. Click the appropriate button in the 'Canvas Options' section to add/remove the height/width to the chosen side.

If you want to clear the grid but retain the size of the grid, click the 'Clear Canvas' option.

There is also an option to 'Toggle Grid Lines' which will remove/restore the grid lines. If you remove it, then you will not be able to add/remove height/width until you restore the grid lines. However, you can still colour the canvas without grid lines.
